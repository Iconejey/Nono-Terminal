Reasoning Process

2. In main.js, runAgentLoop handles tool calls.
3. Specifically, if tool call name is 'execute_command', it calls session.writeCommand().

The agent loop executes commands through a persistent shell session. Here is a high-level summary of the process:

1. **Tool Invocation**: The AI agent decides to call the `execute_command` tool.
2. **IPC Communication**: The command is dispatched to the main Electron process.
3. **Shell Write**: The `ShellSession` class writes the command to the spawned bash process's `stdin`.
4. **Delimiter Tracking**: It appends a delimiter command to capture the exit code and updated working directory.
5. **Output Streaming**: `stdout` and `stderr` streams are captured and sent in real time back to the UI.
